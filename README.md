### 5. Weather App
**Description:**  
An interactive **weather web application** that shows live weather updates based on user location or search query.  
**Key Features:**  
- Current weather information and forecast  
- Integration with live weather API  
- Modern and interactive UI with responsive design  
**Tech Stack:** Python, Flask, API Integration, HTML/CSS, JavaScript  
**Potential Improvements:** Include additional metrics like UV index, humidity trends, or hourly forecast charts.  
**GitHub:** [Weather App](https://github.com/mrjawadd/weather_app)
